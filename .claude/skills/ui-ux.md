# Skill: UI/UX & Style

## Components & Styling
- **Framework:** Tailwind CSS for utility-first styling.
- **Icons:** Lucide-react or Radix Icons.
- **Themes:** Dark/Light mode support using `next-themes` or CSS variables.

## Navigation & State
- **Routing:** File-based routing (e.g., Next.js App Router or React Router).
- **Internationalization (i18n):** `next-intl` or `i18next` for all user-facing text.
- **States:** Always define explicit Loading and Error boundary components.
